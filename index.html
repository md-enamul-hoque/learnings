1. Basic HTML structure:
    <!DOCTYPE html>
    <html>
    <body>
      
    <h1>My First Heading.</h1>
    <p>My first paragraph.</p>
      
    </body>
    </html>

Here, <html></html>, <body></body>, <h1></h1> and <p></p> All are HTML elements. Learning HTML consist of three topics:
* Elements
* Attributes with elements
* Values to attributes
